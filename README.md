# StockBot AO 📦🇦🇴

**Controlo de stock no WhatsApp para cantinas e lojas em Angola.**

Dono recebe alerta no WhatsApp. Funcionário vende com 2 cliques no PWA.

## Stack
- **Backend:** FastAPI + Postgres + SQLAlchemy
- **Frontend Func:** PWA React + Vite
- **Frontend Dono:** Next.js Dashboard
- **Deploy:** Docker + Render

## Como rodar local
```bash
pnpm install
pnpm dev
